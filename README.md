# GIF 制作工具

纯浏览器端 GIF 制作工具，支持多图片合成 GIF 和视频转 GIF。

## 功能

- 多图片合成 GIF（支持 PNG, JPG, WebP, GIF）
- 视频转 GIF（支持 MP4, WebM, AVI, MOV）
- 可调节尺寸、帧率、质量、循环次数
- 视频截取起止段落
- 自动适配图片尺寸（居中裁剪模式）
- 暗色主题现代化界面

## 使用

```bash
npm install
npm run dev
```

## 技术栈

- Vue 3 + Vite 6
- gif.js（GIF 编码）
- @ffmpeg/ffmpeg（视频解码）
